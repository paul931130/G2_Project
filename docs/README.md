# Docs｜詩境文件說明

這裡放置專題報告與作品設計文件。

## 文件列表

1. `PROJECT_REPORT.md`：專題報告，包含專案動機、系統功能、技術架構、版本演進與展示方式。
2. `SOUL.md`：作品人格、互動語氣與設計原則。
3. `UPDATE_SUMMARY.md`：v1.5.0 整合更新、修改檔案、檢查結果與發布前測試清單。

## 根目錄文件

根目錄另有以下文件：

1. `README.md`：GitHub 首頁專案說明。
2. `USER_GUIDE.md`：使用者操作指南。
3. `RELEASE_NOTES.md`：正式版本發布紀錄。
4. `CHANGELOG.md`：完整更新紀錄，內容與 Release Notes 對齊。
5. `CHECKSUMS.txt`：本次發布檔案的 SHA-256 雜湊值。

## 維護規則

- 改功能：更新 `RELEASE_NOTES.md` / `CHANGELOG.md`。
- 改使用流程：更新 `USER_GUIDE.md`。
- 改專題敘述或展示流程：更新 `docs/PROJECT_REPORT.md`。
- 改作品語氣或互動原則：更新 `docs/SOUL.md`。

## v1.5.0 補充

本版補充本機詩庫輸出清理，避免譯文標題、注釋／註解標籤與英譯內容誤出現在前端分析頁籤。另將長篇題序從原詩正文移出，並補齊所有發布說明文件。

## 期末展示新增文件

- `DEMO_VIDEO_SCRIPT.md`：錄影片逐段旁白與展示流程。

- `FINAL_PROJECT_REPORT.docx`：Word 版期末專題報告。
- `DEMO_VIDEO_SCRIPT.docx`：Word 版錄影片腳本。
