# USER_GUIDE｜詩境使用者指南

版本：v1.5.0 release-ready

本指南說明「詩境」網站的主要操作方式。系統可透過本機唐詩資料庫與 Gemini API 流程提供古典詩詞查詢、詩意解讀、朗讀、收藏與互動式對話。沒有 API Key 時，仍可完成主要展示。

## 1. 開始使用

在搜尋欄輸入：

- 詩名，例如：`黃鶴樓送孟浩然之廣陵`
- 作者，例如：`李白`
- 詩句片段，例如：`故人西辭黃鶴樓`
- 意境描述，例如：`離別、思鄉、月夜、孤獨`

系統會優先查詢本機 `poems.json`。若本機詩庫命中，會立即顯示作品；若未命中且已輸入 Gemini API Key，才會進入 AI 查詢流程。

## 2. 詩魂現身

搜尋成功後，畫面會顯示：

- 詩名、作者與朝代
- 原詩正文
- 校注／異文
- 白話釋義
- 情緒與意象分析
- 創作背景
- 詩魂提問區

原詩採直排設計。短詩會集中展示；長詩會讓原文白框依句長垂直伸縮，保留古典詩卷感。

## 3. 校注與資料清理

若資料中包含 `一作`、`又作`、`通` 等異文註記，系統會移到「校注／異文」區，避免混入原詩正文。

白話釋義與背景會清理以下原始標籤：

- 譯文及注釋
- 譯文
- 注釋 / 註解
- 英譯 / English translation

## 4. 隨機、朗讀、收藏與複製

- 隨機召喚：從本機 315 首詩詞中抽取，不消耗 Gemini API。
- 朗讀：使用瀏覽器 Web Speech API。
- 收藏：儲存在目前瀏覽器的 localStorage。
- 複製：複製整理後的詩名、作者與原詩正文。

若瀏覽器停用 localStorage，系統會顯示提示，並建議使用複製功能保留詩作。

## 5. Gemini API Key

點擊左側或上方 `API KEY` 可貼上 Gemini API Key。Key 只會暫存在本分頁 `sessionStorage`，關閉分頁後會清除。

API 狀態包含：

- 僅本機：未輸入 key 或 key 不可用。
- 已連線：Gemini key 輕量測試成功。
- 配額受限：Gemini 回覆 429 或短時間請求過多。

v1.5.0 已使用 Mock Gemini 完成發布驗證，涵蓋 success、429、500 與 timeout，不需要真實 key 也能確認流程。

## 6. 雙魂對話

選擇兩位詩人與主題後，點擊「召喚兩位詩魂」。若 API 可用，會使用 Gemini 生成對話；若 API 不可用，系統會以本機 fallback 生成簡短回應，避免展示中斷。

## 7. 建議展示流程

1. 搜尋 `黃鶴樓送孟浩然之廣陵`。
2. 搜尋 `怨情`，指出輸出清理後不再混入標籤。
3. 搜尋 `琵琶行`，展示長詩直排伸縮。
4. 點擊「隨機召喚」。
5. 使用「朗讀」、「收藏」、「複製」。
6. 切換到「詩藏」查看收藏。
7. 切換到「雙魂對話」，展示 API fallback。
8. 說明本專案已完成三輪以上檢討、修改與發布驗證。

## 8. 常見問題

### 沒有 API Key 可以使用嗎？

可以。本機詩庫支援查詢、隨機、朗讀、收藏、複製與展示。

### 為什麼有些作品查不到？

本機詩庫目前以唐詩為主，未完整收錄宋詞、元曲與所有古典作品。未收錄內容可透過 Gemini API 查詢。

### 為什麼瀏覽器測試不使用真實 Gemini？

為避免暴露 API Key 與消耗配額，發布驗證使用 Mock Gemini。Mock 已覆蓋成功、429 限流、500 錯誤與 timeout。
