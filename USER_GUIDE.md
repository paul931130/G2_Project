# USER_GUIDE｜詩境使用者指南

本指南說明「詩境」網站的主要操作方式。系統可透過本機唐詩資料庫與 Gemini API 提供古典詩詞查詢、詩意解讀、朗讀、收藏與互動式對話功能。

## 1. 開始使用

開啟網站後，可直接在搜尋欄輸入：

- 詩名，例如：`黃鶴樓送孟浩然之廣陵`
- 作者，例如：`李白`
- 詩句片段，例如：`故人西辭黃鶴樓`
- 意境描述，例如：`離別、思鄉、月夜、孤獨`

若本機詩庫中有相符作品，系統會直接顯示詩作內容。若未命中，可透過 Gemini API 進行延伸分析。

## 2. 詩魂現身

搜尋成功後，右側或主要閱讀區會顯示：

- 詩名
- 作者與朝代
- 原詩正文
- 白話釋義
- 情緒與意象分析
- 創作背景或相關補充

原詩區採直排設計，模擬古典詩卷閱讀感。短詩會維持集中展示；長篇作品的原文框會依句長自動垂直伸縮，避免直欄過度擠壓。

## 3. 校注與異文

部分古詩資料會包含異文註記，例如：

```text
唯 通：惟
明年 一作：年年
顰 一作：蹙
```

系統會自動將這些內容從原詩正文移出，放入「校注／異文」區，避免註記混入原文造成閱讀中斷。

## 4. 白話釋義與背景

白話釋義區會自動清理原始資料標籤，例如：

- 譯文及注釋
- 譯文
- 注釋
- 註解
- 英譯

清理後的內容會以較自然的中文段落呈現，使輸出更接近正式閱讀文本。

## 5. 隨機召喚

點擊「隨機召喚」可從本機詩庫中隨機抽取一首作品。此功能不需要 API Key，適合展示或探索詩庫內容。

## 6. 朗讀功能

點擊「朗讀」可使用瀏覽器內建 Web Speech API 朗讀詩文。此功能不依賴外部 TTS 服務，因此不會產生額外 API 成本。

若瀏覽器不支援朗讀，系統會保持其他功能正常運作。

## 7. 收藏與複製

- 「收藏」：將目前詩作儲存在本機瀏覽器。
- 「複製」：複製整理後的詩作內容，方便貼到報告、筆記或簡報。

收藏資料使用 `localStorage` 儲存在使用者裝置，不會上傳到伺服器。

## 8. Gemini API 功能

若輸入 Gemini API Key，可啟用：

- AI 延伸解讀
- 詩魂問答
- 雙魂對話
- 意境式查詢

API Key 僅儲存在目前瀏覽器分頁的 `sessionStorage`，關閉分頁後會自動清除。專案檔案中不包含任何 API Key。

## 9. API 狀態提示

系統會顯示目前狀態：

- 已連線：Gemini API 可用。
- 僅本機：未輸入 API Key，使用本機詩庫。
- 配額受限：API 回傳 429 或短時間無法使用。

即使 API 無法使用，本機詩庫仍可提供查詢、隨機、朗讀、收藏與複製功能。

## 10. 建議展示流程

期末展示可依下列順序操作：

1. 搜尋 `黃鶴樓送孟浩然之廣陵`，展示校注分離與原詩閱讀。
2. 搜尋 `怨情`，展示白話釋義輸出清理。
3. 搜尋 `琵琶行`，展示長詩直排白框伸縮。
4. 點擊「隨機召喚」，展示本機詩庫備援。
5. 使用「朗讀」、「收藏」、「複製」展示互動功能。
6. 若 API Key 可用，展示詩魂問答或雙魂對話。

## 11. 常見問題

### 沒有 API Key 可以使用嗎？

可以。本機詩庫可支援主要查詢、隨機召喚、朗讀、收藏與複製。

### 為什麼有些作品查不到？

本機詩庫目前以唐詩為主，未完整收錄宋詞、元曲與所有古典作品。未收錄內容可透過 Gemini API 查詢。

### 為什麼重新整理後仍看到舊畫面？

系統已加入版本參數降低快取問題。若仍看到舊畫面，可關閉分頁後重新開啟，或等待 GitHub Pages 完成部署。
